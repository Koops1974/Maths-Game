# Cooper's Math Games

A fun, self-contained maths quiz for kids — aimed at KS2/KS3 level (ages 10–13).

## Features

- **5 questions per round** — mix of multiple choice and fill-in
- **Explanations** for wrong answers, showing two methods to solve each problem
- **Per-player stats** — each person gets their own score, history, and leaderboard position
- **Progress Report** — colour-coded topic breakdown so you can see what needs practice (addition, subtraction, multiplication, division, fractions, decimals, percentages, area, powers, time)
- **Persistent scores** — all data saves to browser localStorage
- **Mobile-friendly** — works on phones and tablets

## How to use

1. Open `index.html` in any browser
2. Enter a name and tap **Let's Go!**
3. Answer 5 questions — explanations appear for any wrong answers
4. See your score and review results

No server required — it's a single HTML file. Drop it on any web server or open it directly from the filesystem.

## Hosting on Synology NAS

Install **Web Station** from Package Center, create a virtual host pointing to the folder containing `index.html`, then access it from any device on your network.
